Herramienta para realizar paseos sonoros vinculados al territorio.
Basada en la aplicacion notours de escoitar.org
Usa el editor editor.notours.org para generar el rss del paseo sonoro.
Implemento una aplicación web para disfrutar los paseos en móviles con distintos sistemas operativos.
